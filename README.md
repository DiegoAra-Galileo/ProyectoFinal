# ProyectoFinal
En este repositorio se almacena la documentación y código del sistema de cotizaciones

Flujo principal:

Ciclo principal: El programa se repite hasta que el usuario escriba "Salir"
Recolección de datos:

Nombre del asegurado
Edad (debe ser mayor de 18)
Estado civil
Si está casado: edad del cónyuge (también mayor de 18)
Cantidad de hijos
Cantidad de propiedades
Salario mensual


Cálculo de recargos sobre un precio base de Q2,000:

Por edad del asegurado (10%, 20% o 30%)
Por edad del cónyuge si está casado (10%, 20% o 30%)
Por cada hijo (20% cada uno)
Por cada propiedad (35% cada una)
Por ingresos (5% del salario)


Resultado: Muestra el recargo total y precio final
Repetición: Pregunta si desea otra cotización o salir

En pocas palabras:
Es una calculadora de seguros que suma diferentes porcentajes según las características del cliente y permite hacer múltiples cotizaciones en una sesión.
